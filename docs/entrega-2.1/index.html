Entrega 2.1

Fotogrametria

Primera propuesta de fotogrametria. En este caso de utilizaron 32 fotografias con distintos angulos y perspectivas del plato. Mediante Autodesk Recap Pro se pudo obtener una imagen 3D, con las distintas texturas del plato. En el Sketchfab se puede ver una parte inconclusa del plato, esto ocurrio posiblemente por falta de imagenes que cubrieran esa sección.

Nueva Carpeta (2) by matasolar on Sketchfab

Entrega Final

Esta fotogrametria corresponde a la entrega final de la unidad 2. Para este caso se pestro mas atención a la cantidad de imagenes (54 para este caso) y además de utilizar una iluminación natural y homogena.
